# FF Master Hub

Aplicación profesional para Free Fire: sensibilidades, configuraciones HUD, personajes, armas, mascotas y más.

## 🚀 Funcionalidades
- Selección de idioma al inicio.
- Noticias y agenda semanal.
- Sensibilidades (Normal, DPI, Premium).
- Personajes y habilidades.
- HUD 2/3/4 dedos.
- Sorteos con ID del jugador.
- Ajustes (modo oscuro, idioma, notificaciones).

## 📦 Instalación
1. Clona este repositorio o descarga el .zip.
2. Ejecuta `flutter pub get`.
3. Corre la app con `flutter run`.

## 📌 Requisitos
- Flutter SDK 3.0+
- Android Studio o VS Code con Flutter/Dart plugins instalados.
